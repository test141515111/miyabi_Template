💬 よくある質問

Q1: 毎日続けられるか不安です

A: 最初は1日5分でOK。Daily Noteに「今日の気づき」を1行書くだけでも価値があります。 習慣化のコツは「完璧を目指さない」こと。書けない日があっても大丈夫。

Q2: GitHubは開発者用では？

A: GitHubは「バージョン管理」と「知識管理」のツールです。 Issues = メモ、Discussions = 日記、Labels = タグ として使えば、 Obsidianより強力な知識管理システムになります。

Q3: Obsidianとどっちがいい？

A: Obsidian: ローカルファイル、オフライン、プラグイン豊富
GitHub: クラウド、どこでもアクセス、AI統合、自動化

どちらも素晴らしいツールです。「外出先でもアクセスしたい」「AIと統合したい」ならGitHub。 「完全なプライバシー」「高度なカスタマイズ」ならObsidian。

Q4: 無料で使える？

A: はい、完全無料です！GitHubのPrivateリポジトリは無料で無制限に作成できます。 Cursorも無料プランがあります（月500回のAI質問まで）。
